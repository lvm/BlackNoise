# Black Noise

A tongue-in-cheek UGen that generates Black Noise.

### Installation

Simply open a new document on your SC IDE and type:

    Quarks.install("https://github.com/lvm/BlackNoise");


### Getting started

    {BlackNoise.ar(0.23)}.play;

Pretty much that's it.
